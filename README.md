# Luxuani Textmap Searcher

A search engine for browsing dialogue and text from the Genshin Impact TextMap dataset.
Provides both English and Chinese textmap versions of text spanning over 700,000 text entries.
> **Live Demo:** https://luxuani-textmap-searcher.onrender.com

> **API:** https://luxuani-textmap-searcher-backend.onrender.com/

---

## Features
- Search over 700,000+ TextMap entries
- Search in both English and Chinese
- Highlight matching search terms in English
